=== Subscripts

In vim/nvim, we can insert subscripts for digits from 0 to 9 using digraphs.
kbd:[Ctrl+k] followed by kbd:[<digit> s].
For example, to write N‚Çá, one would type kbd:[N] followed by kbd:[Ctrl+k 7s].

If not using vim, or another subscript is needed, like in maths where we sometimes write sequences like `T1 | T2 | T‚Çô` to indicate that the sequence continues, then the unicode code point can be used.
In the case of ‚Çô, the code point is 0x2299 (hex).
And can be typed in vim with kbd:[Ctrl+v] kbd:[u 2099].

== Bash

Bash accepts the `-e` option to enable parsing of unicode code points:

[source,plain]
----

$ echo -e '\u2714'
